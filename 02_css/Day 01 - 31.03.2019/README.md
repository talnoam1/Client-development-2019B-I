# 3 ways to config a color:
* By color name (ex: `green` , `blue`, etc...)
* By RGB (3 values between 0-255)
```
rgb(255,0,0)		 --> red
rgb(0,255,0)		 --> green
rgb(0,0,255)		 --> blue
rgb(0,0,0)		 --> black
rgb(255,255,255)	 --> white
```

* By RGB (with hexa values - # as prefix + 2 digits for each color)
```
#ff0000		--> red
#00ff00		--> green
#0000ff		--> blue
#000000		--> black
#ffffff		--> white
```
